# NIM Gateway

## Install pnpm

```
npm install -g pnpm
```

## Project setup

```
pnpm install
```

## Compiles and minifies for production

```
pnpm build
```

## Previews the production build

```
pnpm preview
```

## Compiles and hot-reloads for development

```
pnpm dev
```

## Lints and formats files

```
pnpm lint
```
